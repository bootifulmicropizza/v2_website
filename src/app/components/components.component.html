Bootiful micro pizza

fresh from the cloud

Developed by Ian Collington

Can I order some pizza please?


No, sorry. Bootiful Micro Pizza is a fictitious website built by Ian Collington to learn about microservices and cloud technologies.


This website is built using Angular and it is backed by a number of microservices built using Spring Boot/Cloud. It is hosted on AWS in an EKS cluster using Fargate with Istio for the service mesh. Jenkins is also hosted in the EKS cluster to provide a CI/CD environment that deploys straight to production.


The full source code can be found on GitHub.


Whilst you cannot actually order and receive any pizza, this site is otherwise fully functional (or at least will be!) so please go order some pizza!
